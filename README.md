# Catan RL

## Setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Manually step through a game

In one terminal:

```bash
python serve.py
```

In another terminal:

```bash
python main.py setup -v 0
```

# Win Stats Experiment

```bash
python main.py -e win_stats
```
